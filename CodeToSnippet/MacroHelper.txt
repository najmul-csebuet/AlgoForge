/*************************************/
//          Macro Related
/*************************************/

rep: repeat / loop helper
------------------------
rep(${1:i}, ${2:0}, ${3:sz($4)}) {
    $0
}
.

tr: traverse the iterator
-------------------------
tr($1, ${2:a}) {
    $0
}
.

trr: reverse traverse the iterator
----------------------------------
trr($1, ${2:a}) {
    $0
}
.
